# Lucive plugin

Connect your [Lucive](https://lucive.app) Planning account to Claude and
ChatGPT. Read your plan for today, this week, this month, or this year; add,
complete, and reschedule tasks; keep uncommitted week/month/year backlogs
separate from dated commitments; update intents; check your daily scratchpad.

Your Journal is not part of this connection. It stays end-to-end encrypted on
your devices, with no account attached, and cannot be accessed by any agent.

Requires a Lucive account that has been set up in the Lucive app on a
signed-in device.

## Install in Claude

```
/plugin marketplace add lucive-apps/lucive-planning
/plugin install lucive@lucive
```

On first use, the `lucive-planning` server will ask you to connect: sign in
with your email and the 6-digit code Lucive sends you.

## Install in ChatGPT / Codex

```
codex plugin marketplace add lucive-apps/lucive-planning
codex mcp login lucive-planning
```

Then install Lucive from the app's Plugins screen (Personal tab).
